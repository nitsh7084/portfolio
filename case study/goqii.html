The Goqii Theme
The Goqii theme is smart, fun and interactive. Goqii started off with a balanced color scheme with toned down hues. As they started to expand, they felt the need to reflect the fun and interactivity of the Goqii Band. We opted for a little more vibrant and bold secondary colors while keeping the primary color consistent for the Coach App, so that the users still recognize the Goqii brand.
Analytics Insights
Typeface and Color Scheme
We decided to use Gotham as the typeface for the Goqii theme as its thin strokes and proportional characters help create a theme of intelligence. Gotham also contains a number of weights for different text hierarchies. The Goqii Color Scheme is fun and bold. The accent color is a bright and inviting hue useful in making prominent elements and attract attention.
Other Screens
Goqii Home
The illustration on the home screen is a fun and inviting. The home screen provides the coach a quick review of the activity and the progress made by the players assigned to him/her. The Coach can quickly change or assign a goal to any player with the add goal button. The fun colors in the illustration and the welcome message set in a large type size give an inviting feeling to the home screen.
